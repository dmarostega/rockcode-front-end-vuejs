# Roteiro manual - Agrupamento do hub de ferramentas

Issue de origem: <https://github.com/dmarostega/rockcode-front-end-vuejs/issues/42>

## Desktop

1. Acessar `/ferramentas`.
2. Confirmar que existe a seção "Ferramentas para usuários comuns".
3. Confirmar que a "Calculadora de Desconto" aparece nesse grupo.
4. Confirmar que existe a seção "Ferramentas para desenvolvedores".
5. Confirmar que ferramentas como "Gerador de UUID", "Conversor Base64" e "Formatador JSON" permanecem no grupo dev.
6. Confirmar que o QRCodeFlow continua separado em "Projeto relacionado".

## Mobile

1. Validar em largura aproximada de `390px`.
2. Confirmar que os títulos dos grupos, descrições e cards quebram em uma coluna sem sobreposição.
3. Confirmar que os cards continuam clicáveis e apontam para as rotas atuais.
4. Confirmar que não existe rota ou link novo para `/tools`.
