# Roteiro manual - Comparador de preço por unidade

Issues de origem: <https://github.com/dmarostega/rockcode-front-end-vuejs/issues/40>

## Desktop

1. Acessar `/ferramentas/comparador-preco-unidade`.
2. Confirmar H1, link de retorno ao hub, dois blocos de produto e botão de comparação.
3. Informar `22,90`, `5`, `kg` e `5,20`, `1`, `kg`; confirmar custo por kg e melhor opção.
4. Usar exemplos de mercado, farmácia e construção.
5. Trocar uma unidade para categoria incompatível, como `kg` contra `litro`; confirmar erro amigável.
6. Limpar quantidade ou usar `0`; confirmar erro amigável.
7. Confirmar aviso de privacidade, FAQ curta e link relacionado para calculadora de desconto.

## Mobile

1. Validar em largura aproximada de `390px`.
2. Confirmar que inputs, selects, botões e resultado não sobrepõem conteúdo.
3. Confirmar que não há rolagem horizontal.
4. Usar o link "Voltar para ferramentas" e confirmar retorno ao hub.
