# Teste manual - Contador de caracteres e palavras

## Rota

- `/ferramentas/contador-caracteres-palavras`

## Casos rápidos

- Texto vazio: deve mostrar `0` caracteres, `0` palavras, `0` linhas e `0` sem espaços.
- Texto simples: `Rock Code Labs` deve atualizar a contagem em tempo real.
- Texto com quebras de linha: duas linhas preenchidas devem mostrar `2` linhas.
- Texto longo: colar um parágrafo grande não deve travar a página nem quebrar o layout.
- Botões: `Usar exemplo` deve preencher o campo e `Limpar` deve zerar a contagem.

## Smoke visual

- Desktop: validar leitura do H1, área de texto, cards de contagem, conteúdo, privacidade, FAQ e link de retorno.
- Mobile: validar que os cards empilham, botões ocupam largura adequada e não há rolagem horizontal.
