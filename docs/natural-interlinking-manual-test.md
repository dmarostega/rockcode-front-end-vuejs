# Roteiro manual - interlinking natural

## Escopo

- `/`
- `/apps`
- `/ferramentas`
- `/ferramentas/calculadora-desconto`

## Validacao

1. Na home, confirmar que o link "Usar ferramentas gratuitas" leva para `/ferramentas`.
2. Em `/apps`, confirmar que o bloco de ferramentas gratuitas leva para `/ferramentas` e nao aponta para paginas internas de apps ainda nao criadas.
3. Em `/ferramentas`, confirmar que o QRCodeFlow continua apontando para o produto publico atual.
4. Nas ferramentas de compras/economia, confirmar que os links relacionados apontam apenas para ferramentas existentes.
5. Conferir que nenhum bloco de links parece repetitivo, escondido ou criado apenas para SEO.
