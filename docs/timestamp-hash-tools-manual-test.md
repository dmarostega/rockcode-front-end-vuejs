# Roteiro manual - Conversor Timestamp e Gerador de Hash

Issues de origem:

- <https://github.com/dmarostega/rockcode-front-end-vuejs/issues/27>
- <https://github.com/dmarostega/rockcode-front-end-vuejs/issues/28>

## Desktop

1. Abrir `/ferramentas` e confirmar os cards "Conversor Timestamp" e "Gerador de Hash".
2. Abrir `/ferramentas/conversor-timestamp`.
3. Converter `1700000000` como segundos e conferir uma data legível.
4. Converter uma data válida para timestamp e copiar o resultado.
5. Informar timestamp invalido e confirmar erro amigavel.
6. Abrir `/ferramentas/gerador-hash`.
7. Gerar SHA-256 para `Rock Code Labs` e copiar o resultado.
8. Trocar para SHA-512 e confirmar que o hash muda.

## Mobile

1. Usar viewport aproximado de 390x844.
2. Confirmar que inputs, botões, resultados, FAQ e link de retorno não sobrepõem texto.
3. Confirmar que nenhum placeholder/anúncio aparece entre input, botão e resultado.
