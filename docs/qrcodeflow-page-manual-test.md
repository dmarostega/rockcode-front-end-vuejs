# Roteiro manual - pagina interna QRCodeFlow

## Escopo

- `/apps`
- `/apps/qrcodeflow`
- `public/sitemap.xml`

## Validacao

1. Abrir `/apps` e confirmar que o card do QRCodeFlow leva para `/apps/qrcodeflow`.
2. Abrir `/apps/qrcodeflow` e confirmar que existe apenas um `h1` com o nome `QRCodeFlow`.
3. Conferir se a pagina explica problema resolvido, publico-alvo, funcionalidades, stack/status e relacao com a Rock Code Labs.
4. Confirmar que o CTA `Acessar QRCodeFlow` abre `https://qrcodeflow.rockcodelabs.com.br`.
5. Confirmar que os links relacionados apontam apenas para paginas existentes: `/ferramentas` e `/ferramentas/url-encode-decode`.
6. Conferir se o sitemap inclui `https://rockcodelabs.com.br/apps/qrcodeflow/`.
